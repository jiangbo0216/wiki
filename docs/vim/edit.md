# Editing

设置indent样式
:set shiftwidth=2

character | navigation
-|-
a |	Append
A |	Append from end of line
i |	Insert
o |	Next line
O |	Previous line
s |	Delete char and insert
S |	Delete line and insert
C |	Delete until end of line and insert
r |	Replace one character
R |	Enter Replace mode
u |	Undo changes
`<C-R>`	| Redo changes
