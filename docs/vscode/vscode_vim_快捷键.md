## 窗口操作

| 快捷操作 | vim                                                          | vsocde     |
| -------- | ------------------------------------------------------------ | ---------- |
| 分割窗口 | 命令: split, vsplit<br />帮助: `:h split` ,`:h :split`<br /> |            |
| 水平分割 | `CTRL-W_s`                                                   | `CTRL-W_s` |
| 垂直分割 | `CTRL-W_v`                                                   | `CTRL-W_v` |
| 新建空白 | ` CTRL-W_n`                                                  |            |
| 切换窗口 | `CTRL-W_w`                                                   | `CTRL-W_w` |
| 关闭窗口 | `CTRL-W_c`                                                   | `CTRL-W_c` |
| 切换文件 | `CTRL-^`, `ctrl-6`,`ctrl-shift-6`                            | `CTRL-6`   |
| 关闭文件 |                                                              |            |

 

## 区域操作

vscode 对应的快捷键名称为:

view:show

| 区域操作        | vim  | vscode          |
| --------------- | ---- | --------------- |
| 扩展            |      | `CTRL-SHIFT-X`  |
| 文件            |      | `CTRL-SHIFT-E`  |
| debug           |      | `CTRL-SHIFT-D`  |
| search          |      | `CTRL-SHIFT-F`  |
| git             |      | `CTRL-SHIFT-G`  |
| terminal        |      | `CTRL-'`        |
| next plane view |      | `CTRL-P CTRL-V` |
