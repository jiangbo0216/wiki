绳索计算机及其算法
输入: 任给直线L及其上一点A
输出: 经过A做L的一条垂线

算法(勾股定理)
12段等长的绳索,从A点固定4段绳索到B点, 另一个方向伸长3段绳索到C点, 移动点C, 将剩余的3+5段绳索拉直

尺规计算器及其解法
任给平面上线段AB(输入), 将其三等分(输出)

算法:
从A发出一条与AB不重合的射线p,
在p上取AC' = C'D' = D'B'
联接B'B
经D'做B'B的平行线, 交AB与D
经C'做B'B的平行线, 交AB与C

子程序: 过直线外一点,做平行线
